`Rust Constructor` is a GUI library built on `egui`. This tutorial provides a brief overview of how to use `Rust Constructor` in real-world development.

This tutorial is based on `Rust Constructor v2.12.1`. Some features may differ from the version you are using.

This tutorial is designed to help developers get started with `Rust Constructor`. It does not yet cover advanced topics such as tags, page loading, or text selection.
